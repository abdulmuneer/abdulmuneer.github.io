# abdulmuneer.github.io

Source for **Abdul Muneer's personal blog**.

**Live site:** <https://abdulmuneer.github.io/>

Built with [Jekyll](https://jekyllrb.com/) and the
[Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/) theme, deployed via GitHub Actions
(`.github/workflows/pages.yml`).

- `index.md` - blog home (landing page).
- `ignis/ignis.md` - the *Ignis - Finding Your Mojo from DwarfStar* series
  landing page; `ignis/00-*.md` through `ignis/05-*.md` are its six parts.

## Local preview (optional)

```bash
bundle install
bundle exec jekyll serve   # http://localhost:4000
```
